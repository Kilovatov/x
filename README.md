# Multiplication Practice

A simple, browser-based multiplication table quiz built with plain HTML, CSS, and JavaScript — no dependencies or build step required.

## Features

- **Random or selected-table mode** — practice all numbers up to a chosen max, or focus on specific tables
- **Configurable time limit** — set a per-question countdown (3–60 seconds)
- **Max number setting** — choose up to 5, 10, or 12
- **Live score tracking** — correct and wrong counts update in real time
- **Responsive design** — works on desktop and mobile

## Usage

Just open `index.html` in any modern browser:

```bash
open index.html
```

No server, no install, no build process needed.

## How It Works

1. Choose a **Max Number** and **Mode** (Random or Selected Tables)
2. Set a **Time Limit** per question
3. If using Selected Tables mode, toggle which tables to practice using the number buttons
4. Type your answer and press **Check Answer** or hit **Enter**
5. The app tracks your correct and wrong answers as you go
